# VConnect - College/School Club Event Management App

VConnect is a web application designed to help college and school clubs post their upcoming events and allow students to engage by commenting on whether they will participate or not or any adjustments in events that could be made. This app fosters better communication and collaboration between clubs and students.

## Features

- **Club Event Posting**: Clubs can create and share details about their upcoming events.
- **Student Engagement**: Students can comment on events to indicate their interest or participation.
- **User-Friendly Interface**: Simple and intuitive design for both clubs and students.
- **Real-Time Updates**: Events and comments are updated in real-time.

## Tech Stack

- **Frontend**: [React.js](https://reactjs.org/) 
- **Backend**: [Node.js](https://nodejs.org/) with [Express.js](https://expressjs.com/)
- **Database**: [MongoDB](https://www.mongodb.com/) 
- **Version Control**: Git and GitHub

## Installation

1. Clone the repository:
   ```git clone <https://github.com/S-D123/vconnect>```
   ```cd vcon```

2. Install dependencies:
```npm install```

3. Set up environment variables:
Create a .env file in the root directory and add the required environment variables (e.g., database connection string, API keys).

4. Start the development server:
```npm start```

## Usage

1. Clubs can log in and create events with details like title, description, date, and time.
2. Students can browse events and comment on their participation.
3. Admins can manage events and moderate comments.
